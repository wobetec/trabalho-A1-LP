data_info

Esse arquivo contém as listas e dicionários referentes aos dados. Como por exemplo os nomes dos aquivos .csv, as colunas que serão utilizadas e as equivalências entre os códigos e os valores reais do dataset utilizado.

data_work.get_data_ready.remover_linhas_com_valores_em_branco(df: DataFrame, colunas_a_verificar: list)

Parâmetros: df (pd.DataFrame): Dataframe original colunas_a_verificar (list): Lista de colunas a serem verificadas se possuem valores nulos

data_work.get_data_ready.tratar_tipo_dados(df: DataFrame, tipos_validos: list)

Verifica o tipo de dado de cada coluna do dataframe original, se não for do tipo previsto transforma o dado no tipo str

Parâmetros: df (pd.DataFrame): Dataframe original tipos_validos (list): Lista com os tipos de dados que serão aceitos dentro do dataframe
